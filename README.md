# Mastermind

Mastermind with React as frontend and Springboot as backend.

## How to build
### Backend

On Unix:
```
cd backend
./mvnw clean compile package
```

On Windows (not tested):
```
cd backend
mvnw.cmd clean compile package
```

### Frontend
```
cd frontend
npm install
```

## How to execute
### Backend 
On Unix:
```
cd backend
java -jar target/mastermind-0.0.1-SNAPSHOT.jar
```

On Windows (not tested):
```
cd backend
java -jar target/mastermind-0.0.1-SNAPSHOT.jar
```

### Frontend
```
cd frontend
npm start
```
